## 配置

将一个markdown文档变成首页

```yaml
---
home: true
heroImage: /images/logo.png
heroText: '🏷️Hi : )🏷️'
tagline: '静默的Vuepress Blog🏷️'
actions:
  - text: 👉ENTRY
    link: /vuepress/
    type: primary
---
```
