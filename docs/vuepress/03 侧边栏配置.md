## vue侧边栏



## sidebar侧边栏

vuepress支持两种方式的侧边栏

1. 所有页面会使用相同的侧边栏
2. 不同子路径下的页面会使用不同的侧边栏



## 自动生成侧边栏

使用依赖[directory-tree - npm (npmjs.com)](https://www.npmjs.com/package/directory-tree)

```sh
npm i directory-tree
```
